Description

Compatible with iOS 9+

Requires RocketBootstrap greater than v1.0.6 (Add official repo: apt.autotouch.net to install the latest version of RocketBootstrap)

Record and playback touch operations, play macro scripts automatically.

AutoTouch Licenses:

- All licenses have one year validity period which starts from the first activation, not the purchasing time.

- License is transferable: a license can be used to activate another device, meanwhile the last activated device will be diactivated.
